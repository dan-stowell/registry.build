tools

Functionality and tooling used in bzlparty projects.

Overview

This package provides several toolchains and rules that might be useful to set up bzlparty projects.

Toolchains

The toolchains bundled into this module:

  • biome
  • fd
  • goawk
  • ripgrep
  • sha
  • shellcheck
  • shellspec
  • templ
  • typos
